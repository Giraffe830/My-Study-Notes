node.js是一个事件驱动、非阻塞式的I/O模型，但是单线程，一个基于 Chrome V8 引擎的 JavaScript 运行时环境

作用: 它允许 JavaScript 代码在浏览器之外执行（即在操作系统层面上运行）。
它提供了文件系统读写 (fs)、网络通信 (http/net)、进程管理等底层 API。

1.用户发送 HTTP 请求到达 Node.js 的路由层，路由解析后调用控制器（Controller）分析功能，node请求连接数据库
2.node利用数据库池连接，目的是复用可以节省每次建立连接的时间
3.node向数据库发送请求，node注册此请求事件
4.非阻塞挂起，异步等待，数据库处理数据，node主线程去管其他的HTTP请求
5.数据库返回数据，执行注册的回调函数，触发业务逻辑，释放连接，但池管理器不关闭 TCP 连接，处理队列中的请求
