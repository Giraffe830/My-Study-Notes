npm run dev 就是启动 Vite 的开发模式。

运行在RAM内存里
----------------------------------------------------------------------------------------------------
npm run build 是启动 Vite 的生产打包模式。
1、翻译
vite把所有文件翻译成浏览器看得懂的.js和.css；
2、压缩
它会把 函数方法的名字改成 f()。
它会把你写的注释全部删掉。
目的： 让文件体积变小，用户打开网站的速度就像闪电一样快。
3、优化。
假设你引入了一个很大的工具包有 100 个功能，但只用了其中 1 个。
Vite 会像摇晃一棵树一样，把那 99 个没用的枯叶子扔掉，不打包进去。
