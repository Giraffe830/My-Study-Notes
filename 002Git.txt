Git 是一个运行在本地的、分布式的版本控制工具（负责记录历史、回滚、分支）。

而 GitHub 是一个基于 Git 的云端托管平台（负责存储代码、协助多人协作）。
我们用 Git 管理代码，用 GitHub 分享和同步代码。
---------------------------------------------------------------------------------
git add .暂存
旁边是不是有个 + (加号)？
点击那个加号。
会发现文件从 “更改 (Changes)” 这一栏，到了 “暂存的更改 (Staged Changes)” 这一栏。
含义： 这就是把文件扔进快递箱了。
------------------------------------------------------------------------------------------------------------
git commit -m "这里是提交文本"提交
在上面的 消息框 (Message) 里，输入备注：这里是提交文本。
点击下面的 蓝色大按钮 "提交 (Commit)"。
含义： 快递箱封口，生成历史记录。
------------------------------------------------------------------------------------------------------------
git push同步
提交完后，蓝色按钮变成了 "同步更改 (Sync Changes)"（或者在左下角有个循环箭头）。
点击它。VSCode 会自动帮你把代码传到 GitHub。
